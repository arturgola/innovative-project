**acsess hsy cache that server is created:** http://localhost:3000/hsy-cache

**env file example in backend**

```
# OpenAI Configuration
OPENAI_API_KEY=

# HSY Waste Guide API Configuration
HSY_CLIENT_ID=
HSY_CLIENT_SECRET=

# Server Configuration
PORT=3000
```

**to run backend:**

```
node server.js
```

**to run frontend:**

```
npx expo start
```
